# 📊 TA效能报告 - 自动化分析工具

医疗健康行业 **招聘效能（TA Effectiveness）** 数据分析自动化工具。  
基于 LLM Wiki 架构，支持批量上传公司调研问卷，自动生成行业级 P25/P50/P75 分位数分析报告。

---

## 🏗️ 系统架构

采用三层 **LLM Wiki** 架构：

```
📥 Raw Data（原始数据层）
   ↓  问卷上传 → 自动解析 Excel
🔍 数据验证与清洗
   ↓  范围校验 / 单位修正 / 加总一致性 / 逻辑检查
📚 Wiki（知识库层）
   ↓  结构化知识条目 → 多公司聚合
📊 Scheme（技能执行层）
   ↓  26个分析维度 → 指标透视 → 报告撰写
📄 行业报告输出
```

### 核心模块

| 模块 | 说明 |
|------|------|
| `app.py` | Streamlit 前端界面（单公司 / 多公司模式） |
| `multi_company.py` | 多公司数据聚合 + 行业报告生成器 |
| `run_single_company.py` | 单公司问卷解析引擎 |
| `data_validator.py` | 数据清洗与验证（基于核查总表规则） |
| `raw_data/` | Raw Data 层 — 问卷摄入、列名映射 |
| `wiki/` | Wiki 层 — 知识库、编译器 |
| `scheme/` | Scheme 层 — 分析维度、指标透视、报告撰写、校验 |

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动应用

```bash
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501`

### 3. 使用流程

1. **选择模式**：左侧边栏选择「单公司分析」或「行业报告（多公司）」
2. **上传问卷**：上传 Excel 格式的原始公司调研问卷
3. **自动分析**：系统自动解析、验证、生成报告
4. **下载结果**：支持下载 Markdown 报告、CSV 数据、JSON 知识库

## 📋 支持的分析维度（26个）

### 招聘量指标
- 各职能招聘量占比（BY 不同职能 / 不同规模公司）
- 各职级招聘量占比（BY 不同职级 / 不同规模公司）

### 招聘渠道指标
- 各招聘渠道招聘量占比（HR直招 / 猎头 / 内推 / 内部转岗 / RPO / 校招）
- 各职能下不同招聘渠道占比
- 各职级下不同招聘渠道占比

### 招聘周期指标
- 各职能招聘周期（天）
- 各职级招聘周期（天）
- 不同规模公司交叉分析

### 招聘成本指标
- 不同职能人均招聘成本（万元）
- 渠道成本分布（猎头费 / 内推费 / RPO费）
- 不同规模公司成本对比

### 细分职能报告
- 商业二级职能细分（渠道、周期、成本）
- 研发二级职能细分（渠道、周期、成本）

### TA生产率
- TA人均招聘量
- TA团队配置

## 📁 问卷文件要求

上传的 Excel 文件应为 **原始公司调研问卷**（非汇总表），包含以下 Sheet：

| Sheet名 | 内容 |
|---------|------|
| 基本信息 | 公司名称、规模等 |
| TA 公司整体效率分析 | 各职能/职级的招聘量、渠道、周期、成本 |
| TA部门效率分析-研发 | 研发细分职能数据 |
| TA部门效率分析-商业 | 商业细分职能数据 |
| TA 人员配置 | TA团队FTE配置 |
| 综合招聘效能 | 问卷定性数据 |

> ⚠️ **注意**：请上传各公司的原始问卷文件（如 `2024TA BeiGene_xxx.xlsx`），而非汇总表文件（如 `4.1 TA公司整体效率汇总表.xlsx`）

## 🔍 数据验证规则

系统内置了多层数据验证：

1. **单位校正**：自动识别并修正单位不一致（如罗氏的元→万元）
2. **范围校验**：招聘周期1-365天、人数非负等（成本字段无上限）
3. **加总一致性**：公司整体 vs 各职能加总差异检查（>5%报警告）
4. **逻辑一致性**：人均成本异常高（>50万/人）标记关注
5. **零值处理**：有招聘量但周期为0自动排除

## 📊 输出说明

- **行业报告**（Markdown）：包含 P25/P50/P75 分位数和平均值
- **汇总数据**（CSV）：所有公司扁平化数据表
- **知识库**（JSON）：结构化知识条目

> 当某维度样本量 n≤2 时，P25/P50/P75 值可能相同，表格中会以 ⚠️ 标注
