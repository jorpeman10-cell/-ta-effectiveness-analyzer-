# TA效能报告自动化工具：报告结构与数据口径说明

本文档说明当前工具生成报告的结构、数据提取口径、分析维度，以及 2025 年调研问卷、上一年度最终口径表与历史 PDF 报告之间的使用边界。

## 1. 项目定位

本工具用于医疗健康行业 TA 效能调研数据的自动化读取、清洗、聚合与报告生成。

当前业务场景为：

- 数据来源：2025 年 17 家公司提交的原始调研问卷 Excel。
- 报告用途：用于准备 2026 年发布的 TA 效能报告。
- 上年度对比数据：优先使用上一年度最终口径表 Excel（例如 `医药行业TA数据服务项目报告_表格整理_修正版.xlsx`），不再依赖 PDF/PPTX 文本抽取或中间处理总表推算。
- 历史 PDF：仅作为 Sheets 3/4/5 分析维度和呈现口径参考。

重要口径：

历史 PDF 中的数据不参与 2025 年问卷的任何比例、分位数、样本量或结论计算。年度效能对比的上年度数据来自最终口径表 Excel；当年数据来自 2025 年 17 家原始问卷。

## 2. 工具入口与核心模块

| 模块 | 作用 |
|---|---|
| `app.py` | Streamlit 前端入口，支持单公司、行业报告、年度对比分析 |
| `multi_company.py` | 读取公司效能数据，生成行业级 TA 效能报告 |
| `survey_parser.py` | 读取问卷 Sheets 3/4/5，生成招聘实践趋势分析 |
| `yoy_comparator.py` | 年度对比报告生成，支持当年问卷 vs 上年发布报告 |
| `report_parser.py` | 解析上一年度最终口径表 Excel；保留中间总表、PDF/PPTX 解析作为备用 |
| `data_validator.py` | 数据校验、清洗、异常提示 |

## 3. 数据来源与提取口径

### 3.1 2025 年原始问卷

工具读取各公司提交的原始 Excel 问卷，主要包括：

| Sheet | 读取内容 |
|---|---|
| 基本信息 | 公司名称、公司规模分类等 |
| TA 公司整体效率分析 | 公司整体及一级职能的招聘量、渠道、周期、成本 |
| TA 人员配置分析 | TA FTE、团队配置、TA 人均招聘量 |
| 综合招聘效能提升 | Sheet 3，招聘实践、候选人体验、雇主品牌等定性题 |
| 热点职位调研 | Sheet 4，2026 新增 HC 预测和热点职能 |
| 高管任期变化趋势调研 | Sheet 5，高管任期缩短趋势及企业应对 |

### 3.2 公司规模口径

公司按规模分为：

| 类别 | 口径 |
|---|---|
| A 类公司 | 公司人数大于等于 1500 人 |
| B 类公司 | 公司人数小于 1500 人 |

报告中的整体、A 类、B 类口径均基于问卷中的公司规模信息进行聚合。

### 3.3 指标计算口径

核心数值指标采用公司级数据聚合后计算：

- P25、P50、P75：基于有效样本公司的指标分布计算。
- 平均值：基于有效样本公司算术平均。
- 占比类指标：先在单家公司内计算占比，再对公司占比分布取 P50 等分位数。
- 招聘周期：使用有效招聘周期数据，空值或明显无效值不参与分位数。
- 成本：统一按万元口径处理，校验阶段会识别明显单位异常。

### 3.4 Sheets 3/4/5 定性问卷口径

Sheets 3/5 中大量答案存储在 Excel 表单控件中，而不是普通单元格。当前工具会读取 Excel 文件内部的 VML 表单控件，识别单选/多选勾选项。

定性题统计口径：

- 每家公司每道题作为一个有效样本。
- 单选题按选项分布计算比例。
- 多选题按选项被选择次数占总选择次数计算比例。
- 每道题展示该题实际有效响应公司数，不强制等于 17。
- A 类/B 类分布基于有效响应公司的规模分类分别统计。

## 4. 报告输出结构

### 4.1 行业 TA 效能报告

由 `IndustryReportGenerator.generate_full_report()` 生成，主要结构为：

1. 整体概览
2. TA 效能报告
3. 附录：人力效益指标列表

核心分析内容包括：

| 模块 | 分析维度 |
|---|---|
| 招聘量 | 公司整体招聘量、各一级职能招聘量占比、A/B 类公司差异 |
| 招聘渠道 | HR 直招、猎头、内推、内部转岗、RPO、主动投递、校招等渠道分布 |
| 招聘成本 | 不同职能人均招聘成本、猎头费、RPO 费、内推费等成本结构 |
| 招聘周期 | 各职能、各职级招聘周期 P25/P50/P75 |
| 细分职能 | 商业、研发等二级/三级职能的渠道、周期、成本表现 |
| TA 生产率 | TA FTE、TA 人均招聘量、团队配置效率 |
| TA 人员配置 | 公司整体、COE function、TA BP、各职能 TA FTE 与第三方 TA/RPO 配置 |

### 4.2 年度对比报告

由 `YoYReportComparator.generate_yoy_report()` 生成，适用于：

- 当年数据：2025 年 17 家调研问卷。
- 上年数据：上一年度最终口径表 Excel。

报告结构：

1. 招聘周期趋势
2. 招聘成本趋势
3. 渠道分布趋势
4. TA 生产率趋势
5. 细分职能趋势
6. 成本结构趋势
7. 关键发现
8. TA 招聘实践趋势分析（Sheets 3/4/5）
9. 数据提取审核

注意：

年度对比报告中的数值趋势对比优先使用上一年度最终口径表 Excel；Sheets 3/4/5 的招聘实践趋势分析只使用 2025 年问卷数据，历史 PDF 只提供维度参考。

### 4.3 Sheets 3/4/5 招聘实践趋势分析

由 `SurveyReportGenerator.generate_pdf_aligned_trend_report()` 生成。

标题口径：

`2026报告TA招聘实践趋势分析（基于2025年调研问卷）`

数据声明：

`2024年数据/2025年发布PDF仅用于参考分析维度和呈现口径，不参与任何比例、样本量或结论计算。`

分析结构：

| 章节 | 数据来源 | 参考口径 |
|---|---|---|
| 招聘策略与候选人识别 | Sheet 3 | 参考历史报告中的岗位招聘策略、价值观匹配、核心能力识别维度 |
| 候选人体验、Offer 接受与雇主品牌 | Sheet 3 | 参考历史报告中的 Offer 接受、雇主品牌、候选人体验维度 |
| 招聘流程前置介入与权限边界 | Sheet 3 | 参考历史报告中的流程介入、权限边界、数字化工具维度 |
| 2026 新增 HC 预测与热点岗位 | Sheet 4 | 基于 2025 问卷填写的 2026 HC 预测和热点职能 |
| 高管任期变化趋势 | Sheet 5 | 基于 2025 问卷中的高管任期变化观察、原因和应对策略 |

## 5. Sheets 3/4/5 具体分析维度

### 5.1 Sheet 3：综合招聘效能提升

Sheet 3 覆盖以下模块：

| 模块 | 示例问题方向 |
|---|---|
| Employer Branding | 是否实施雇主品牌建设、如何吸引人才、雇主品牌带来的影响 |
| Talent Acquisition Strategy | 岗位差异化招聘策略、价值观匹配、核心能力与长期潜力识别 |
| Recruitment Process Optimization | 面试流程标准化、流程介入节点、权限边界、技术工具使用 |
| Candidate Experience Management | 候选人流程体验、反馈收集、面试体验改善 |
| Performance Management | 招聘策略评估、招聘 KPI、流程瓶颈识别、渠道效果评估 |
| Talent Retention and Development | Offer 接受率提升、薪酬福利竞争力、职业发展机会 |
| Hitting Rate | Hitting Rate 变化方向、上升/下降职能及原因 |
| Cost for per hiring | 招聘成本包含的人群与费用口径 |

### 5.2 Sheet 4：热点职位调研

Sheet 4 主要用于 2026 年报告中的前瞻性内容：

- 2026 财年新增 HC 预测。
- 公司整体及职能维度的新增 HC。
- 重点关注/热点职能 Top 排名。
- 热点职能关注比例。

当前原始问卷中部分热点职能通过表格单元格或勾选项体现，工具会尽量读取原始文件；若存在已编制汇总表，也支持读取汇总表补充分析。

### 5.3 Sheet 5：高管任期变化趋势调研

Sheet 5 主要用于观察高管岗位服务周期变化：

| 模块 | 示例问题方向 |
|---|---|
| 高管任期现状感知 | 是否观察到高管服务周期缩短、集中岗位、平均实际任期 |
| 驱动因素分析 | 内部管理压力、外部市场变化、组织转型等因素 |
| 组织与人才影响 | 高管更替对组织稳定、业务连续性、团队信心的影响 |
| 企业应对策略 | 选聘标准调整、入职后支持、继任计划、文化适配 |
| 未来展望与建议 | 是否成为新常态、未来 1-2 年趋势、理想任期、外部支持需求 |

## 6. 数据校验与异常处理

工具内置以下校验：

- Excel 文件可读性校验。
- 基本信息缺失检查。
- 招聘量、周期、成本字段的数值化处理。
- 公司整体与职能加总一致性检查。
- 招聘周期合理范围检查。
- 成本单位异常识别。
- 空值、零值、无效样本自动排除。
- Sheets 3/5 表单控件读取失败时，不影响主效能报告生成，但该题样本数会相应减少。

## 7. 当前样本读取状态

基于目前提供的 17 家 2025 年原始问卷文件，工具验证结果如下：

| 数据块 | 当前读取状态 |
|---|---|
| 主 TA 效能数据 | 进入行业报告和年度对比报告 |
| Sheet 3 | 多数公司可读取有效响应，单题样本数按实际响应计 |
| Sheet 4 | 17 家进入读取，热点职能/HC 按可读字段统计 |
| Sheet 5 | 多数公司可读取有效响应，单题样本数按实际响应计 |

报告中所有样本数均以实际读取成功且该题有效响应的数据为准。

## 8. 对外说明建议

对客户或报告读者建议使用以下表述：

> 本章节基于 2025 年 17 家参调企业问卷数据生成，用于 2026 年 TA 效能报告。历史发布报告仅用于延续分析框架和维度口径，不参与本年度问卷题目的数据统计。

> 定性问卷题目按有效响应公司数统计，不同题目的样本数可能不同；多选题结果反映选项被选择的相对集中度。

## 9. 当前调整记录

详细调整记录见 [TOOL_ADJUSTMENTS.md](TOOL_ADJUSTMENTS.md)。

当前已确认的关键调整包括：

- 报告结构需要补充招聘量分析。
- 主报告内容顺序调整为：数据口径、招聘量、渠道、成本、成本结构、周期、细分职能、TA 生产率、Sheet 4 前瞻、关键发现、Sheet 3、Sheet 5、总结、数据审核。
- Sheet 4 内容前置，用作 2026 新增 HC 与热点岗位前瞻导读。
- Sheet 3 与 Sheet 5 不再合并在一个综合章节中，而是各自形成独立报告章节。
- TA 生产率章节需要同时解释 TA 配置、第三方 TA 支持、COE function / 系统化能力。
- 工具会读取并输出 TA 人员配置数据，包括公司整体、COE function、TA BP、早期研发、临床开发、商业、生产及供应链、职能等维度下的 TA FTE 与第三方 TA/RPO。
- 0 值需要 trim，除非业务确认该 0 值代表真实发生。
- PPT 生成模块暂作为实验工具保留，当前视觉风格未确认，不自动生成正式 PPT。
